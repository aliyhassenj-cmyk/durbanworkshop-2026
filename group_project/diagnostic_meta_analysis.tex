\subsection{Overview}\label{overview}

This document performs a full diagnostic accuracy meta-analysis on the
data in \texttt{data/Diagnostic\_Accuracy\_Table.csv} using the
\texttt{mada} and \texttt{meta} packages.

We'll cover:

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  Load \& clean data
\item
  Sensitivity / Specificity with heterogeneity (I²)
\item
  Subgroup analysis (by diagnostic type)
\item
  Diagnostic odds ratio (DOR)
\item
  Forest plot
\item
  SROC curve
\item
  Funnel plot \& publication bias test
\end{enumerate}
